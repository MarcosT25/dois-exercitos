Para compilar e executar o programa siga os seguintes passos: 
1. Entrar na pasta "sources": 
```bash
cd sources/
```
2. Compilar todos arquivos cpp:
```bash
g++ -c *.cpp
```
3. Unir os arquivos compilados em um executável chamado "app":
```bash
g++ *.o -o app 
```
4. Rodar o app:
```bash 
./app
```
